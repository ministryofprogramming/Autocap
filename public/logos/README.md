# AutoCap Logos

Brand assets for the AutoCap application.

## Available Logos

- `autocap-white.png` - White/transparent version (for dark backgrounds)
- `autocap-dark.png` - Black/red transparent version (for light backgrounds)

## Usage in Next.js

```tsx
import Image from 'next/image'

// Dark backgrounds
<Image src="/logos/autocap-white.png" alt="AutoCap" width={200} height={50} />

// Light backgrounds
<Image src="/logos/autocap-dark.png" alt="AutoCap" width={200} height={50} />
```

## Public URL

These files are accessible at:
- `/logos/autocap-white.png`
- `/logos/autocap-dark.png`
