# Reference Documentation

Source materials and reference documents for the AutoCap project.

## Files

### `website-copy-deck.docx`
Complete website copy and content structure. Use this as the source of truth for all website text, messaging, and content hierarchy.

### `feature-teardown.xlsx`
Detailed product feature breakdown and analysis. Contains feature specifications, priorities, and technical details.

### `data-map.csv`
Workshop location data including:
- 13 acquired workshop locations across Sweden
- Geographic coordinates (lat/long)
- Acquisition timeline (2025-2026)
- Contact information and websites
- Regional distribution (primarily Stockholm and Västra Götaland)

## Usage

These files are reference materials for:
- Content creation
- Feature planning
- Data modeling
- Business context

When implementing features, refer to these documents for accurate information.
